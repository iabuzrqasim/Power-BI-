# 📊 Amazon Clothing Sales 2025 — Power BI Dashboard

An interactive Power BI report analyzing **Amazon Clothing Sales data** across regions, product categories, brands, customer segments, and delivery performance.

---

## 🗂️ Report Pages

### 1️⃣ Sales Overview
High-level summary of overall business performance.

| Visual | Type | Description |
|--------|------|-------------|
| Total Revenue | KPI Card | Overall revenue generated |
| Total Orders | KPI Card | Total number of orders placed |
| Total Products | KPI Card | Number of unique products |
| Average Rating | KPI Card | Customer satisfaction score |
| AOV | KPI Card | Average Order Value |
| Revenue by Main Category | Bar Chart | Revenue breakdown by product category |
| Revenue by Region | Map | Geographic distribution of sales |
| Orders by Payment Method | Pie Chart | Payment preference analysis |
| Revenue Trend | Line Chart | Revenue performance over time |
| Brand Table | Table | Top performing brands |
| Customer Age Group Table | Table | Orders by age segment |

---

### 2️⃣ Product, Category & Brand Analysis
Deep dive into product performance across categories and brands.

| Visual | Type | Description |
|--------|------|-------------|
| Total Revenue | KPI Card | Page-level revenue metric |
| Total Orders | KPI Card | Page-level order count |
| Total Products | KPI Card | Product count |
| Average Rating | KPI Card | Avg product rating |
| AOV | KPI Card | Average Order Value |
| Revenue by Brand & Category | Bar Chart | Brand-wise revenue split by category |
| Quantity Sold by Sub-category | 100% Stacked Bar | Sales volume share across sub-categories |
| Avg Rating by Sub-category | 100% Stacked Bar | Rating distribution across sub-categories |
| Brand Detail Table | Table | Detailed brand-level breakdown |
| Slicers | Filters | Main Category, Age Group, Region |

### 3️⃣ Review & Delivery Insights
Customer experience, device usage, and delivery performance.

| Visual | Type | Description |

## 🛠️ Tools & Technologies

- **Tool:** Microsoft Power BI Desktop
- **Data Source:** Amazon Clothing Sales 2025 (`Amazon_Clothing_Sales_2025`)
- **File Format:** `.pbix`

## 📐 Key Measures Used

| Measure | Description |
|---------|-------------|
| `Total Revenue` | Sum of all sales revenue |
| `Total Orders` | Count of all orders |
| `Total Product` | Count of distinct products |
| `Average Rating` | Mean customer rating |
| `AOV` | Average Order Value = Revenue / Orders |
| `Avg Product Rating` | Average rating per product |
| `Total Quantity Sold` | Total units sold |
